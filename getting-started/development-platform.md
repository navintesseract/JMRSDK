# Development Platform

JMRSDK supports following Mixed Reality development environments.

### Android Environment

* Android 9
* Android 10
* Android 11
* Android 12
* Android 13

### iOS Environment

iOS 11 or higher

### Unity3D Versions

{% hint style="success" %}
**Recommended Version: 2021.3.19f1**
{% endhint %}

* 2022.x
* 2021.x
* 2020.x
* 2019.4.16f1+

{% hint style="info" %}
This documentation is based on JMRSDK used with Unity **2021.3.19f1**
{% endhint %}
