# Development Platform

JMRSDK supports following Mixed Reality development environments.

### Android Environment

* Android 9
* Android 10
* Android 11
* Android 12
* Android 13
* Android 14

### iOS Environment

iOS 11 or higher

### Unity3D Versions

* 2021.3.x

{% hint style="info" %}
This documentation is based on JMRSDK used with Unity **2021.3.xf1**
{% endhint %}
