# Development Platform

JMRSDK supports following Mixed Reality development environments.

### Android Environment

* Android 11
* Android 12
* Android 13

### Unity3D Versions

{% hint style="success" %}
**Recommended Version: 2019.4.40f1**
{% endhint %}

* 2019.4.16f1+
* 2020.x
* 2021.x

{% hint style="info" %}
This documentation is based on JMRSDK used with Unity 2019.4.40f1
{% endhint %}
